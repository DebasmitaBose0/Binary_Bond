## ğŸš€ Quick Start

### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Binary-Bond/binarybond-v1.0-portfolio.git
   cd binarybond-v1.0-portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist/` directory.

### Preview Production Build

```bash
npm run preview
```

### Adding New Projects

1. Open `src/components/Projects.jsx`
2. Add new project object to the `projects` array
3. Include: title, description, techStack, team, github, status, category

2. Update `src/components/Contact.jsx` for contact information
3. Modify social links and profiles as needed

- **Fonts**: Modify `index.html` for Google Fonts integration
- **Animations**: Adjust Framer Motion settings in components
