### Prerequisites

- Node.js (version 16 or higher)

### Installation

1. Clone the repository
```bash
git clone <your-repo-url>
cd binarybond-v1.0
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm run dev
```

4. Open your browser at `http://localhost:5173`

### Build for production

```bash

```

The built files will be in the `dist/` directory.

To preview the production build locally:

```bash
npm run preview
```

## ğŸ”§ Customization

### Adding New Projects

1. Open `src/components/Projects.jsx`
2. Add new project object to the `projects` array
3. Include: title, description, techStack, team, github, status, category

### Updating Team Information

1. Edit `src/components/Team.jsx` for team member details
2. Update `src/components/Contact.jsx` for contact information
3. Modify social links and profiles as needed

### Styling Changes

- **Colors**: Update `tailwind.config.js` for custom color schemes
- **Fonts**: Modify `index.html` for Google Fonts integration
- **Animations**: Adjust Framer Motion settings in components
